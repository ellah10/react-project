Recipe Sharing Platform

A simple web application that allows users to share, view, and manage recipes.
Users can create accounts, upload recipes with images, and interact with content from others.

Features:
- Authentication: Sign up, login, and logout with Firebase Authentication.
- Recipe CRUD: Add, edit, delete, and view recipes.
- Image Upload: Upload recipe images to Firebase Storage.
- Comments: Allow user comments on recipes.
- Favorites : Users can save favorite recipes.
- Routing: Navigate between pages using React Router.

Technologies & Dependencies

- React: Frontend UI library
- TypeScript: Static type checking
- Firebase: Authentication, Firestore database, and Storage
- React Router DOM: Page navigation and routing
- CSS Modules or SCSS: Styling
- Vite (or Create React App): Project setup and development server

To install main dependencies:
npm install react react-dom firebase react-router-dom
npm install --save-dev typescript @types/react @types/react-dom
